# Personal Portfolio Website

This is a clean, modern, and responsive portfolio website to showcase your skills, projects, and contact information.  
It is designed to help you present yourself professionally during interviews or when applying for internships.

---

## Features

- Responsive layout suitable for desktop and mobile devices  
- Smooth scrolling navigation  
- About Me section with profile picture and bio  
- Skills section with stylish skill badges  
- Projects showcase with images, descriptions, and GitHub links  
- Contact form with basic validation and user feedback  
- Clean UI with a modern color palette and typography
